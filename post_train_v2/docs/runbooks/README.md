# Runbooks

Operational commands for implemented V2 phases live here.

- `data_and_evaluation.md`: source data, fixed splits, Teacher generation, and
  common evaluation.
- `supervised_and_rft.md`: Full SFT, LoRA SFT, RFT rollout, and RFT training.
- `dpo.md`: DPO data construction and training.
- `grpo.md`: verl GRPO conversion, launch, selection, and export.
- `full_pipeline.md`: end-to-end orchestration.
- `recovery.md`: resume and rebuild behavior.
- `final_evaluation.md`: final model matrix evaluation.
- `acceptance.md`: local and remote acceptance gates.
