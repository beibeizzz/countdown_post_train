# DPO Data

Generated by:

```bash
python post_train/scripts/dpo/build_dpo_data.py --config post_train/configs/dpo_data.yaml
```

Important outputs:

- `dpo_train.jsonl`: chosen/rejected preference pairs.
- `dpo_rejected_candidates.jsonl`: full rejected candidate debug log.
- `manifest.json`: category counts, route counts, malformed and wrong-value fractions.
