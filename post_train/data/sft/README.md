# SFT Data

Generated by split and RFT scripts.

Common commands:

```bash
python post_train/scripts/data/build_sft_splits.py --config post_train/configs/data_build.yaml
python post_train/scripts/sft/build_rft_data.py --config post_train/configs/rft.yaml
```

Important outputs:

- `sft_train_8k.jsonl`: main full/LoRA SFT train data.
- `rft_accepted.jsonl`: accepted RFT sampled responses.
- `manifest.json`: split counts and seeds.
