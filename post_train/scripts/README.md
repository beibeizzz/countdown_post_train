# Scripts

Command-line entrypoints for the Countdown post-training pipeline.

Subdirectories:

- `data`: source build, teacher pool, split builders.
- `sft`: full SFT, LoRA SFT, RFT data and RFT training.
- `dpo`: DPO data construction and training.
- `grpo`: GRPO training.
- `eval`: standalone model evaluation.
