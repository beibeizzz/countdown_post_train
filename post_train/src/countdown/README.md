# Countdown Utilities

Shared modules:

- `prompts.py`: canonical prompt builders.
- `generation.py`: chat-template and vLLM generation wrappers.
- `solver.py`, `validation.py`: exact Countdown solving and answer validation.
- `bucketing.py`, `sampling.py`: stratified sampling support.
- `eval.py`: common sample and aggregate evaluator.
- `io.py`, `config.py`: file and config helpers.
